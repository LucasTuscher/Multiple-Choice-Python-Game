# Lern-Quiz

Multiple-Choice-Quiz zum Lernen von Signalverarbeitung und Computergrafik.

## Installation

```bash
# Virtual Environment aktivieren
cd "/Users/lucastuscher/Desktop/Lern Code Spiele"
source .venv/bin/activate

# CustomTkinter installieren (falls noch nicht geschehen)
pip install customtkinter
```

## Starten

### GUI (empfohlen)
```bash
python Multiple-Choice/gui.py
```

### Konsolen-Version
```bash
python Multiple-Choice/main.py
```

## Features

- Moderne GUI mit CustomTkinter (funktioniert auf macOS, Windows, Linux)
- Themenauswahl: Signalverarbeitung, Computergrafik
- Mehrfachauswahl bei Fragen moeglich
- Ausfuehrliche Erklaerungen nach jeder Antwort
- Fortschrittsanzeige und Score-Tracking
